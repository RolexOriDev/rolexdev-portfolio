## Idea 1: "Terminal Noir" — Dark Brutalist Developer Aesthetic

**Design Movement:** Dark Brutalism meets Terminal UI
**Core Principles:**
1. Monospace typography as a design element — code IS the art
2. High-contrast dark canvas with neon accent (electric cyan or acid green)
3. Deliberate asymmetry — sections break the grid intentionally
4. Raw, unpolished edges that feel authentic to a developer's world

**Color Philosophy:**
- Background: Deep charcoal (#0d0d0d)
- Accent: Electric cyan (#00e5ff) — evokes terminal cursor, code energy
- Text: Off-white (#e8e8e8) for readability
- Secondary: Muted slate (#3a3a4a) for cards

**Layout Paradigm:**
- Left-anchored sidebar navigation (fixed, minimal)
- Full-width hero with animated typing effect
- Projects in a masonry-style grid with hover reveal

**Signature Elements:**
1. Blinking cursor animation on hero title
2. Code-snippet decorations as background texture
3. Neon glow on hover for interactive elements

**Interaction Philosophy:** Every interaction mimics a terminal command — hover reveals "run" states, clicks trigger "execute" animations

**Animation:** Typewriter on load, slide-in from left for sections, glitch effect on name

**Typography System:** JetBrains Mono (headings) + IBM Plex Sans (body)

## Idea 2: "Precision Engineering" — Swiss Minimalism with Technical Depth

**Design Movement:** Swiss International Typographic Style meets Modern SaaS
**Core Principles:**
1. Grid discipline — every element aligned to an invisible 8px baseline grid
2. Typography as hierarchy — size and weight carry all meaning
3. Monochromatic base with a single surgical accent color
4. Whitespace as a luxury signal — breathing room = confidence

**Color Philosophy:**
- Background: Near-white (#f8f7f4) — warm, not clinical
- Foreground: Near-black (#111111) — strong, decisive
- Accent: Deep indigo (#3d3df5) — intellectual, technical, trustworthy
- Muted: Warm gray (#9a9a9a) for secondary text

**Layout Paradigm:**
- Top navigation bar with name left-anchored, links right
- Hero: large typographic statement, left-aligned, no image
- Skills: horizontal scrolling tag cloud
- Projects: 2-column editorial cards with large numbering

**Signature Elements:**
1. Large ordinal numbers (01, 02, 03) as section markers
2. Thin horizontal rules as section dividers
3. Underline animations on links (draw-in effect)

**Interaction Philosophy:** Calm and precise — hover states are subtle color shifts, no jarring transforms

**Animation:** Fade-up on scroll (Framer Motion), staggered list reveals, smooth page transitions

**Typography System:** Space Grotesk (headings, bold) + DM Sans (body, regular)

## Idea 3: "Obsidian Code" — Dark Elegant with Glassmorphism Accents

**Design Movement:** Dark Luxury meets Developer Craft
**Core Principles:**
1. Deep dark background with glass-card layering for depth
2. Gradient accents — electric blue to violet — for energy without chaos
3. Asymmetric hero with floating code block decoration
4. Clean sans-serif typography with tight tracking for professionalism

**Color Philosophy:**
- Background: #0a0a0f — near-black with blue undertone
- Card surface: rgba(255,255,255,0.04) glassmorphism
- Accent gradient: #4f8ef7 → #8b5cf6 (blue to violet)
- Text: #f0f0f5 primary, #8888aa secondary

**Layout Paradigm:**
- Full-screen hero with name large left, floating animated code snippet right
- Sticky top nav with blur backdrop
- Skills as animated progress bars or icon grid
- Projects: horizontal scroll carousel on mobile, 3-col grid on desktop

**Signature Elements:**
1. Glassmorphism cards with subtle border glow
2. Gradient text on key headings
3. Floating particle or grid-dot background animation

**Interaction Philosophy:** Fluid and immersive — cards lift on hover with shadow bloom, buttons have gradient shimmer

**Animation:** Particle background (canvas), scroll-triggered fade-up, gradient shimmer on CTA

**Typography System:** Outfit (headings, bold) + Geist (body, clean)
